## âœ… Implementation Details

### 1. IPO List Page
- Implemented using a **responsive table layout** instead of cards
- Displays the following information as required:
  - Company name
  - Issue date
  - Issue size
  - Price range
  - Minimum investment
  - Number of shares and lots
- Each row is clickable and navigates to the IPO Details page
- Horizontal scrolling enabled on smaller screens for better data readability

**Why Table Layout?**
- IPO data is structured and column-based, making tables more readable
- Ensures consistent alignment of financial data
- Scales well for larger datasets

---

### 2. IPO Details Page
- Home navigation placeholder
- Download button for IPO-related documents
- Displays complete IPO information
- Fully responsive design:
